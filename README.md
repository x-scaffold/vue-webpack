# x-scaffold-vue-admin

> 根据`vue init webpack`生成的模块修改，支持多页面配置


## 安装 ##

### 项目初始化 ###

``` bash
$ vue init x-scaffold/vue-admin my-project
```

### 安装依赖 ###

``` bash
$ npm install
```

**ps:** 如果出现权限问题，加`sudo`

### 预览 ###

``` bash
$ npm run dev
```

### 项目构建 ###

统一项目路径

``` bash
$ ln -s ~/YourProjectWorkpace/** ~/ProjectRoot/
```

``` bash
$ npm run build
```

### 项目构建支持生成图表依赖分析 ###

```bash
$ npm run build --report
```

### run unit tests ###

```bash
$ npm run unit
```

### run e2e tests ###

```bash
$ npm run e2e
```

### run all tests
```bash
$ npm test
```
